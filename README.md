# SnowSeeker
학습용 SwiftUI 토이프로젝트입니다.

# 사용된 기술
- NavigationView
  - Screen size에 따른 2가지 타입
- Environment value 
  - dynamicTypeSize
  - sizeClass (horizontal, vertical)
- alert
  - 옵셔널 값을 전달받는 alert문
